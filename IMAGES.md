## Cara Download Gambar untuk Mockup

1. **Copy URL** dari daftar di atas
2. **Paste di browser** untuk melihat gambar
3. **Klik kanan > Save Image As** untuk download
4. Atau **gunakan parameter** di URL untuk custom size:
   - Ganti `w=1080` dengan ukuran yang diinginkan (contoh: `w=2000` untuk kualitas lebih tinggi)
   - Ganti `q=80` dengan quality (max: `q=100`)

## Contoh Custom URL (High Quality)
```
https://images.unsplash.com/photo-1629353689974-af4d5c70440f?q=100&w=2000&fm=jpg
```

## Tips untuk Mockup:
- Semua gambar dari Unsplash bebas untuk commercial use
- Ukuran original bisa sampai 4K-6K pixels
- Format: JPG (untuk mockup cetak/digital)
- Aspect ratio: Portrait/Square (cocok untuk product display)

## Brand Colors W2U:
- **Primary**: Yellow #FFC107 (rgb: 255, 193, 7)
- **Secondary**: Dark Gray #1F2937 (rgb: 31, 41, 55)
- **Background**: Gray-900 #111827 (rgb: 17, 24, 39)
- **Text**: White #FFFFFF untuk dark bg
- **Accent**: Gray-50 #F9FAFB untuk light bg

## Typography:
- **Font Family**: System font stack (Inter, sans-serif)
- **Logo**: Bold, 3xl size
- **Tagline**: Small, 0.75rem
